# ADR-002: Always draft first, approval required before publish

**Status:** Accepted

**Context:**
Publishing errors are hard to reverse. WordPress posts can be deleted but Dev.to syndication with canonical URL affects SEO immediately.

**Decision:**
All generation produces a draft JSON with `status: draft`. Publish tools check for `status: approved` before executing. No publish path bypasses the approval gate. Approver field records who approved.

**Consequences:**
Nothing publishes without explicit approval. Future review agent sets approver field automatically. Human approval is default for MVP.
